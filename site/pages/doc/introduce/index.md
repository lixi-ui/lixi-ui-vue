## 介绍

### 引入

```sh 
  # 安装
  npm install lixi-ui-vue
```

```js
  // 代码中使用 例子
  import { Button } from 'lixi-ui-vue';
```

#### web的介绍
```js
  // web 通常是用于业务逻辑开发
```

#### ui的介绍
```js
  // 提供一个 ui 库, 主流框架上的组件都会有.
```

#### material的介绍
