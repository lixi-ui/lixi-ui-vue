## 介绍

### 引入

```sh 
  # 安装
  npm install lixi-ui-vue
```

```js
  // 引入
  import lixiUiVue from 'lixi-ui-vue';
  var app = createApp(App).use(lixiUiVue);
```

```html
  <!-- 项目中使用 -->
  <lx-button></li-button>
```

### cdn 的使用
```js
  <script src="https://unpkg.com/lixi-ui-vue@0.0.6/lib/index.js"></script>
```
